# Todo App

A simple multi-project Todo application built with JavaScript, Webpack, and ES6 Modules.

## Features

- Create Projects
- Create Todos
- Mark Todos as Complete
- Delete Todos
- Persistent Data using Local Storage

## Technologies Used

- HTML
- CSS
- JavaScript
- Webpack

## What I Learned

- Modular JavaScript architecture
- ES6 imports and exports
- DOM manipulation
- State management
- Object-Oriented Programming (OOP)
- Local Storage
